## 📖 Descripción del Proyecto
Este proyecto forma parte de la *Actividad 4.1: Control de Versiones (Git & GitHub)* y tiene como objetivo principal aprender y aplicar los conceptos fundamentales del control de versiones utilizando **Git** y **GitHub**.

A lo largo de esta actividad, trabajaremos de forma colaborativa para gestionar cambios en el código, mantener un historial de versiones y coordinar el trabajo en equipo de manera eficiente.

---

## 🎯 Objetivos
- Comprender el funcionamiento básico de **Git**.
- Aprender a usar repositorios en **GitHub**.
- Gestionar versiones de un proyecto.
- Trabajar de forma colaborativa mediante ramas (*branches*).
- Resolver conflictos de fusión (*merge conflicts*).

---

## 🛠️ Tecnologías utilizadas
- Git
- GitHub
- Terminal / Línea de comandos
