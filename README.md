# Calculadora en Java

Este proyecto es una calculadora básica desarrollada en Java que permite realizar operaciones matemáticas simples como suma, resta, multiplicación y división.

## 📌 Funcionalidades

- Sumar dos números
- Restar dos números
- Multiplicar dos números
- Dividir dos números con control de división por cero
- Menú interactivo por consola

## 🛠️ Tecnologías usadas

- Java
- Scanner (entrada por consola)
- JUnit 5 (para pruebas unitarias)
